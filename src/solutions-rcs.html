  • PC & Server
  • IP Camera(CCTV)
  • VNC Server
  • DVR / NVR
  • PC 기반 장비, PLC Touch Screen (GOT, Pro-Face Remote S/W PC) (키오스크, 반도체 검사장비 등)

생산공정 통합관리

Display, 반도체, 자동차등 제조기업의 생산공정을 안정적으로 원격 제어 및 통합 모니터링 하는 솔루션입니다.

고객중심 커스터마이징

독자적인 원격제어 및 통합 모니터링 기술을 기반으로 고객이 요청한 커스터마이징을 적극 수행하는 사용자 중심의 솔루션입니다.
